Privacy Policy

Last updated: June 19, 2026

1. Introduction

GoReturn ("we", "our", "us") is a Shopify application that helps merchants manage product returns and exchanges. This Privacy Policy explains how we collect, use, and protect information when you use our app.

2. Information we collect

When you install and use GoReturn, we collect:

  • Store information: Your Shopify store domain, store name, and email address provided during OAuth authorization.
  • Order data: Order details including order IDs, customer names, email addresses, product information, and order amounts — accessed via Shopify API with your permission.
  • Return request data: Information submitted through return requests including customer name, email, phone number, product details, return reason, and refund preferences.

3. How we use your information

We use collected information solely to:

  • Provide return and exchange management services
  • Process and track return requests
  • Facilitate refunds through Shopify's built-in refund system
  • Send notifications about return status updates
  • Display analytics and reporting in your dashboard

4. Data sharing

We do not sell, rent, or share your personal data or your customers' data with third parties. Data may be shared only with:

  • Shopify: To process refunds and access order information via Shopify API
  • Shipping partners: Only when you explicitly connect a shipping aggregator (e.g., Shiprocket) for return pickup scheduling
  • Messaging services: Only when you explicitly enable WhatsApp or email notifications

5. Data storage and security

Your data is stored securely on Railway-hosted PostgreSQL databases with SSL encryption. We implement industry-standard security measures including encrypted connections, secure API authentication, and access controls.

6. Data retention

We retain your data for as long as you have the app installed. Upon uninstalling GoReturn, your store data and return records will be deleted within 30 days. You can request immediate deletion by contacting us.

7. Your rights

You have the right to:

  • Access your data stored in our systems
  • Request correction of inaccurate data
  • Request deletion of your data
  • Export your return data
  • Revoke app access by uninstalling from Shopify

8. Shopify API usage

We access your Shopify store data through Shopify's official API with the following permission scopes:

  • read_orders, write_orders: To view orders and process refunds
  • read_customers: To access customer information for return requests
  • read_products: To display product details in return forms
  • read_inventory: To check stock availability for exchanges

9. Cookies

Our application does not use tracking cookies. We only use essential session data required for app functionality.

10. Changes to this policy

We may update this Privacy Policy from time to time. We will notify you of significant changes through the app dashboard or email.
